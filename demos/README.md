# Demos

This folder tracks demo plans, scripts, captures, and results.

## First demo

Goal:

> Same real place. Same geometry. Switch between neutral, 1812, and 3055.

## First demo script

1. Open app.
2. Show neutral AR camera view.
3. Detect floor/wall/road surface.
4. Tap and label surface.
5. Select `1812` Reality Pack.
6. Show older-world visual reinterpretation.
7. Select `3055` Reality Pack.
8. Show future-city visual reinterpretation.
9. Record screen or OBS capture.

## Demo quality bar

The viewer should understand the concept without a long explanation.

The demo does not need:

- perfect object recognition
- glasses
- AI-generated every-frame visuals
- city-scale mapping
- multiplayer
